# trSoft_interview

### Project Launch

Clone repository:
```bash
git clone https://github.com/Mopkovka13/trSoft_interview.git
```

Go to folder:
```bash
cd trSoft_Interview
```

Setup and launch containers
```bash
docker-compose up --build
```

Start application ;)
